# GoPokemon
Game built with OpenGL and GLUT, the goal is to dodge the pokeballs. It's the opposite of PokemonGo.
 - OBJ Reader
 - AABB collision
 - 3D transforms(Scale, rotation, translate) using matrices
 - Generate Bezier
 - Illumination using Phong model
